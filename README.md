# Stream

Exemplo de stream com operações intermediárias e terminais.

São operações intermediárias:
• filter
• map
• flatmap
• peek
• distinct
• sorted
• skip
• limit (*)

São operações terminais:
• forEach
• forEachOrdered
• toArray
• reduce
• collect
• min
• max
• count
• anyMatch (*)
• allMatch (*)
• noneMatch (*)
• findFirst (*)
• findAny (*)
* short-circuit
